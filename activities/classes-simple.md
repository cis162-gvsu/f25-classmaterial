### Question 1:
Suppose you wish to create a `Contact` class
representing a single contact on your phone.
This contact should have a first name, last name,
and phone number.

The contact may also have (but not necessarily)
birthday, address, and email.

Create the constructor for this class.

### Question 2
Suppose you have a file of the form:
```
firstname,lastname,phone,birthday,address,email
```

Write a function to read in this file and
create an instance of a `Contact` for each entry.
